## Implemented

- Added `agentfs integrity <id-or-path> --json`
  - Runs `PRAGMA integrity_check`
  - Validates required tables/config
  - Checks inline/chunk storage invariants
  - Checks namespace, symlink, and optional overlay metadata invariants
  - Exits nonzero on failure

- Added `agentfs backup <id-or-path> <target.db> --verify`
  - Checkpoints/truncates WAL
  - Copies only the main DB file
  - Reopens and verifies the backup when `--verify` is passed
  - Refuses existing targets
